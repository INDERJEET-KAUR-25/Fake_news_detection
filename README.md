# 📰 Fake News Detection using Machine Learning & NLP

## 📌 Overview

This project is a machine learning-based web application that detects whether a news article is **real or fake** using Natural Language Processing (NLP).

## 🚀 Features

* Classifies news as **Real or Fake**
* Uses NLP techniques for text preprocessing
* Implements **Logistic Regression & Naive Bayes**
* Interactive web app using Streamlit
* High accuracy (~95%)

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* TF-IDF Vectorizer
* Streamlit

## 📊 Machine Learning Models

* Logistic Regression
* Multinomial Naive Bayes

## ⚙️ How It Works

1. Text cleaning and preprocessing
2. Feature extraction using TF-IDF
3. Model training
4. Prediction via web interface
